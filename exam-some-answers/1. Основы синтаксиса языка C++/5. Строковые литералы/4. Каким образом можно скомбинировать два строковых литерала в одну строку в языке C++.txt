строки можно складывать, это называется конкатенацией.
конкатенация строковых литералов:
---------------------------------
const char* str = "это очень длинная строка, "
                  "которая была разбита на несколько "
                  "линий для удобства чтения";		// она просто слепится
const char str2 = "Hello" ", " "World" "!";

конкатенация - это объединение. свойство - не ассоциативна:
-----------------------------------------------------------
std::string s1 = "баран";
std::string s2 = "Акакий";
std::string s3 = s1 + s2;
std::string s4 = s2 + s2;
#include <cassert>
assert(s3 != s4);
std::cout << "successfully\n";

так как const char* не поддерживает +, то можно провернуть следующий трюк:
--------------------------------------------------------------------------
const char str1[] = { 'H', 'e', 'l', 'l', 'o', '\0'};
const char str2[] = { ',', ' ', 'W', 'o', 'r', 'l', 'd', '!', '\0'};

std::string str3 = str1;
str3.append(str2);		// метод append(); добавляет строку str2 в конец строки str3
std::cout << str3 << " \n";
