строковый литерал - это последовательность символов в двойных кавычках

можно объявлять так:
const char* str = "Hello, World!";
const char str[] = "Hello, World";
auto str = "Hello, World!";		// автоматически const char*

можно так:
const char str1[] = { 'H', 'e', 'l', 'l', 'o', '\0'};
const char str2[] = { ',', ' ', 'W', 'o', 'r', 'l', 'd', '!', '\0'};

коды символьных литералов внутри строкового литерала можно посмотреть, если
навести курсор на идентификаторы - str1[] или str2[]
